# 🎬 Termux Video Editor Tool (ETit)

অ্যান্ড্রয়েড ইউজারদের জন্য টার্মাক্স ভিত্তিক একটি সুপার-ফাস্ট ভিডিও স্লাইসিং এবং এডিটিং টুল। এটি বিশেষভাবে লো-এন্ড ডিভাইস (যেমন: ৪জিবি র‍্যাম এবং পুরনো অ্যান্ড্রয়েড ভার্সন) এর জন্য অপ্টিমাইজ করা হয়েছে।

## 🚀 Features (বৈশিষ্ট্যসমূহ)

- **Bullet Speed Slicing:** কোনো রি-এনকোডিং ছাড়া সরাসরি 'Stream Copy' মেথডে ভিডিও স্লাইস করা হয়, যা কয়েক সেকেন্ডের মধ্যে কাজ শেষ করে।
- **Low-End Friendly:** ৪জিবি বা তার কম র‍্যামের ফোনেও ক্রাশ করবে না এবং প্রসেসরের ওপর চাপ কমাবে।
- **Dual Mode:** - **Mode 1 (QuickSplit):** কাস্টম কাটিং সুবিধা।
    - **Mode 2 (Auto Slicer):** একটি ভিডিওকে সমান কয়েক ভাগে অটোমেটিক ভাগ করা।
- **Web Dashboard:** ফাইল আপলোড করার জন্য একটি ফ্ল্যাস্ক (Flask) ভিত্তিক ওয়েব ইন্টারফেস।
- **Auto-Update:** প্রতিবার টুলটি রান করার সময় এটি অটোমেটিক গিটহাব থেকে লেটেস্ট আপডেট চেক করে নেয়।
- **User-Friendly UI:** টার্মিনালে কোনো অক্ষর ভেঙে যাবে না এবং ভিডিওর সঠিক ডিউরেশন (mm:ss) দেখার সুবিধা থাকবে।

🎮 How to Use (ব্যবহার পদ্ধতি)
​১. টার্মিনালে শুধু run লিখে এন্টার দিন (যদি সেটআপ করা থাকে)।
২. টুলটি শুরুতেই গিটহাব থেকে লেটেস্ট আপডেট চেক করে নেবে।
৩. মেনু থেকে আপনার পছন্দের মোড সিলেক্ট করুন (যেমন: Auto Slicer এর জন্য 2)।
৪. clips/ ফোল্ডারে আপনার ভিডিওগুলো রাখুন অথবা ওয়েব ইন্টারফেস (http://127.0.0.1:5000) ব্যবহার করে আপলোড করুন।
৫. ভিডিও নম্বর দিন এবং কয়টি ভাগে ভাগ করতে চান তা লিখুন।
৬. আপনার এডিট করা ভিডিওগুলো ready_video/ ফোল্ডারে জমা হবে।
**📂 Project Structure**
​sc.py: মেইন কন্ট্রোল প্যানেল, অটো-আপডেটার এবং ওয়েব সার্ভার।
​tools/: ভিডিও এডিটিং এর সব পাইথন স্ক্রিপ্ট (যেমন: AutoSlicer.py)।
​clips/: যেখানে ইনপুট ভিডিওগুলো থাকবে।
​ready_video/: আউটপুট ফাইলগুলোর লোকেশন।
​app/: ওয়েব ড্যাশবোর্ডের এইচটিএমএল এবং সিএসএস ফাইল।
