## 🚀 বৈশিষ্ট্য (Features)
* **Web Interface:** ফ্লাস্ক সার্ভারের মাধ্যমে ব্রাউজারে ভিডিও আপলোড।
* **Auto Slicer:** ভিডিওকে স্বয়ংক্রিয়ভাবে ছোট ছোট ক্লিপে ভাগ করা।
* **Custom Cutter:** নিজের পছন্দমতো সময় সেট করে ভিডিও কাটা।
* **Easy Automation:** এক কমান্ডেই গিটহাবে প্রজেক্ট আপডেট করার সুবিধা।
